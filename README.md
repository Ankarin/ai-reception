# ai-reception

AI receptionist — Next.js + Clerk + AI SDK (Google), MCP, Drizzle/Neon.

**Live:** https://ai-reception-rouge.vercel.app

## Stack

- TypeScript
- Next.js
- React
- Drizzle ORM
- Neon Postgres
- Clerk auth
- oRPC
- Tailwind
- shadcn/ui
- Vercel AI SDK

## Run

```bash
bun install
bun run dev
```
